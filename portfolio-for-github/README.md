## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- Yarn or npm

### Installation
```bash
# Clone the repository
git clone [your-repo-url]

# Navigate to project directory
cd portfolio

# Install dependencies
yarn install

# Start development server
yarn start
```

### Build for Production
```bash
yarn build
```
